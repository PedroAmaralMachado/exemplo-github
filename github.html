Introdução github e git

Github é uma plataforma de repositorios de projetos.

Enquanto o git é uma ferramenta para controle de versionamento.

O github tem a linha cronologica de main e branch, onde o main é onde ocorre as funcionalidades principais do projeto enquanto o branch é a linha cronologica de pequenas funcionalidades.

O git tem comandos como de commit para divulgar os codigos escolhidos do projeto no repositorio e outros comandos de push, pull, remote e add o push para colocar no repositorio novos codigos ou novas alterações, o pull para puxar do repositorio o projeto e por fim existe o remote para definir a origem de onde está o repositorio e o add para adicionar os arquivos/codigos que devem ser commitados.
